ОПИСАНИЕ
SUIF (Секретная Уникальная Информация Файла) — это утилита 
для шифрования файлов. Приложение превращает любой 
выбранный файл в формат .suif, который невозможно 
расшифровать без соответствующего ключа.

КАК ИСПОЛЬЗОВАТЬ
1. Запустите encryptor.exe.
2. Введите свой Secret Key в поле ввода (если оставить 
   пустым, приложение создаст или использует файл secret.key).
3. Нажмите кнопку "Выбрать файл и зашифровать".
4. Файл будет зашифрован и сохранен в ту же папку с 
   расширением .suif.

ВАЖНАЯ ИНФОРМАЦИЯ О БЕЗОПАСНОСТИ
- Ключ шифрования (secret.key) является единственным способом 
  доступа к вашим данным.
- Если вы удалите файл secret.key или потеряете ключ, 
  зашифрованные файлы .suif станут недоступны навсегда.
- Для максимальной защиты рекомендуется хранить ключ 
  на отдельном физическом носителе (флешке).

ТРЕБОВАНИЯ
Приложение работает на ОС Windows. Никаких дополнительных 
установок не требуется.
